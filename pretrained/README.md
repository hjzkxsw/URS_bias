## Pretrained models 

Please download the unified checkpoint and specialist checkpoints from Google Drive:

```text
https://drive.google.com/drive/folders/11d_Ot9hMOeDRAENP9XtgH_H3n1UhvYHO?usp=sharing
```

Place them under:

```text
./pretrained/
```

The default unified model path is:

```text
./pretrained/unified_checkpoint_500.pt
```

Specialist checkpoints follow this pattern:

```text
./pretrained/{problem}_checkpoint_300.pt
```

For example:

```text
./pretrained/tsp_checkpoint_300.pt
./pretrained/cvrp_checkpoint_300.pt
./pretrained/acvrp_checkpoint_300.pt
```
